DirectX Features of Microsoft® Windows API Code Pack for Microsoft®.NET Framework

This release of Microsoft Windows API Code Pack for .NET contains managed APIs for the following DirectX features:

-          DXGI 1.0 and 1.1 – Most APIs

-          Direct3D 11.0, 10.1 and 10.0 – Most APIs

-          Direct2D 1.0 – Most APIs

-          DirectWrite – Subset of the APIs

-          Windows Imaging Component (WIC) - Subset of the APIs

Windows API Code Pack contains the following components to support DirectX development in .NET:

-          A managed solution containing wrappers for DirectX interfaces, APIs, structs and enumerations.

-          Sample code and helper utilities.

-          Help file (DirectXCodePackHelp).

The code DirectX managed library is provided in the form of a Visual C++/CLI library that requires Visual Studio 2008 with Microsoft Visual C++ installed on it.
The DirectX solution is provided here: <code pack root>\WindowsAPICodePack\DirectX\DirectX.sln.

The DirectX samples and utilities are provided under  <code pack root>\Samples\DirectX.
Samples are provided for both VB.NET and C# programming languages.

 

 Building the library and samples:

To build the DirectX solution, you need:

-          Visual Studio 2008 with the following components installed

o   Visual C++: for the managed library and sample utilities.

o   Visual C#: for the C# samples.

o   Visual Basic: for the VB samples.

-          Windows SDK for Windows 7 installed on your system, only C++ headers and libraries components are needed.

o   Visual Studio C++ paths for the headers correctly referencing the Windows 7 SDK “include” path.

-          DirectX SDK (optional):

o   Most samples can be built using Windows 7 SDK only, however a few samples have dependencies on DirectX SDK.
These samples are:

§  Samples\DirectX\CS\Direct3D10\Tutorial09\Tutorial09.sln

§  Samples\DirectX\VB\Direct3D10\Tutorial09\Tutorial09.sln

§  Samples\DirectX\CS\Direct3D10\MeshBrowser\MeshBrowser.sln

§  Samples\DirectX\VB\Direct3D10\MeshBrowser\MeshBrowser.sln

§  Samples\DirectX\CS\Direct3D10\TextureSwap\TextureSwap.sln

§  Samples\DirectX\VB\Direct3D10\TextureSwap\TextureSwap.sln

 

Running sample solutions:

Samples can be run on Windows 7 or Windows Vista SP2 with Platform Update for Windows Vista installed.

However, the samples mentioned in the DirectX dependencies section above, also require DirectX SDK installed on the target machine (or at least the required redistributable dlls).

 

64-bit Considerations:

To unify the sample code, all library and samples included are pre-configured to be built and run on x86.

To build a 64-bit version of the library and/or the samples, you'll need to have the required 64-bit support components installed with Visual Studio 2008. Please refer to the references below for additional information on how to install Visual Studio 64-bit components.
